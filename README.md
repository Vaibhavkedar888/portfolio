# Vaibhav's Portfolio

A responsive personal portfolio website featuring animations, project showcases, and a contact form.

## Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop.
- **Interactive UI**: Includes parallax effects, typing animations, and scroll reveals.
- **Project Showcase**: Interactive cards with hover effects to display work.
- **Contact Form**: Integrated form handling with notification system.

## Technologies Used

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (ES6+)

## Setup
Simply open `index.html` in your web browser to view the portfolio.
